Projects

Security Knights

Introduction: A gamified secure development training program for developers across the organization, structured around medieval ranks like Paige, Squire, and Knight.

Description: This initiative was designed to embed security culture into developer workflows by using engaging learning paths, hands-on challenges, and progress-based recognition.

Objective: To enhance secure coding awareness and reduce common vulnerabilities across the SDLC by training developers in an interactive, memorable way.

Process: I created course content based on OWASP Top 10, ran live secure code review sessions, collaborated with engineering leads, and gathered feedback to iterate each level.

Tools and Technologies Used: GitHub, Microsoft Teams, PowerPoint, Lucidchart, Google Forms, JavaScript (for demo code), OWASP Juice Shop.

Value Proposition: Boosted security posture by empowering developers to identify and mitigate vulnerabilities early, improving the organization’s DevSecOps maturity.

Unique Value: Themed approach and gamified learning helped increase participation and long-term retention of secure coding principles.

Relevance: Highly relevant in today's cybersecurity climate, where developer-led security is essential for scalable secure software delivery.

References: Internal training feedback forms (redacted), OWASP.org

PromptCraft

Introduction: A curated AI prompt marketplace optimized for model performance and creative use cases.

Description: PromptCraft connects prompt engineers, creatives, and businesses with high-quality, optimized prompts for different AI tools and LLMs.

Objective: To provide a centralized space for tested prompts, user feedback, analytics, and marketplace-style prompt access.

Value Proposition: Saves users time by removing guesswork, while offering monetization for prompt creators.

Unique Value: Focus on optimization per AI model and transparent performance metrics set it apart from generic prompt dumps.

Relevance: As prompt engineering becomes a career path, PromptCraft is positioned as a premium, structured platform.

References: OpenAI documentation, prompt engineering communities, early user surveys
