Objective

Design a platform that helps users create high-quality AI prompts from raw ideas, optimize them with structured frameworks, and test them live against real AI models.

🔴 Problem

Most people type vague one-liners and get inconsistent AI responses. There's no tool that coaches you through building a structured, reusable prompt from scratch.

🟢 Solution

PromptCraft takes your rough idea, applies Role + Context + Constraints + Output engineering, scores it, and lets you test the result live — turning guessing into a predictable system.

Design Process
01
Research

Identified that 80%+ of AI users write single-sentence prompts, leading to inconsistent outputs. Competitors focus on libraries, not engineering.

02
Framework Design

Built the Role + Context + Constraints + Output (RCCO) prompt engineering framework as the core product mechanic.

03
Live API Integration

Connected Claude API to allow instant real-world testing of optimized prompts — showing the before/after difference in real outputs.

04
Scoring System

Designed a prompt quality score that evaluates specificity, structure, and completeness to give users measurable feedback.

05
UX Iteration

Tested with non-technical users. Key insight: people needed examples to understand what "rough idea" meant — added 6 quick-load examples.
