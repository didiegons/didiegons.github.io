Machine Learning Algorithms: Visual Framework

This infographic organizes major machine learning algorithms by learning style, domain relevance, and real-world use. It highlights how traditional supervised and unsupervised methods differ from deep learning approaches used in computer vision, natural language processing, and generative AI.

From General Categories to Specific Algorithms

Machine Learning

Systems that learn patterns from data in order to predict, classify, group, or generate outputs.

Supervised Learning

Uses labeled data to learn direct relationships between input features and known outputs.

Linear Regression

Supervised 📊 Tabular Data

How it works: Fits a line that models the relationship between input variables and a numeric outcome.

Use case: Predicting house prices or sales revenue.

Logistic Regression

Supervised 📊 Tabular Data 💬 NLP

How it works: Estimates the probability that an input belongs to a category.

Use case: Spam detection or loan approval prediction.

Decision Tree

Supervised 📊 Tabular Data

How it works: Splits data into branches using if-then rules until it reaches a final decision.

Use case: Credit risk analysis or fruit classification.

Random Forest

Supervised 📊 Tabular Data

How it works: Combines many decision trees and votes on the final prediction.

Use case: Fraud detection and customer churn prediction.

Support Vector Machine

Supervised 📊 Tabular Data 🖼️ Computer Vision

How it works: Finds the best boundary that separates classes with the widest margin.

Use case: Image categorization and medical diagnosis support.

Unsupervised Learning

Works with unlabeled data to discover hidden patterns, groups, or lower-dimensional structure.

K-Means Clustering

Unsupervised 📊 Tabular Data

How it works: Divides data into clusters by grouping points around central means.

Use case: Customer segmentation and product grouping.

Hierarchical Clustering

Unsupervised 📊 Tabular Data

How it works: Builds a tree of nested clusters by repeatedly merging or splitting data points.

Use case: Gene analysis and similarity grouping.

Principal Component Analysis (PCA)

Unsupervised 📊 Tabular Data 🖼️ Computer Vision

How it works: Reduces many features into fewer components while preserving important variation.

Use case: Data visualization and preprocessing before modeling.

Deep Learning / Modern AI

Uses layered neural networks to learn complex representations from images, text, audio, and sequence data.

Convolutional Neural Network (CNN)

Deep Learning 🖼️ Computer Vision

How it works: Learns visual patterns by scanning images for shapes, textures, and spatial features.

Use case: Face recognition, X-ray analysis, and object detection.

Recurrent Neural Network (RNN / LSTM)

Deep Learning 💬 NLP

How it works: Processes sequences one step at a time while remembering earlier information.

Use case: Time-series forecasting and earlier language models.

Transformer

Deep Learning 💬 NLP ✨ Generative AI

How it works: Uses attention mechanisms to understand relationships between all parts of an input.

Use case: ChatGPT, translation, summarization, and text generation.

How the Algorithms Relate

  • Supervised algorithms learn from labeled data and are strong for prediction and classification.
  • Unsupervised algorithms discover patterns without labels, making them useful for clustering and dimensionality reduction.
  • Deep learning models can handle more complex inputs such as images, language, and generative tasks.
  • Classical methods are often easier to interpret, while deep learning usually performs better on large, unstructured data.

Application Domains

📊 Tabular Data Used for forecasting, fraud detection, customer analytics, and structured business data.
🖼️ Computer Vision Used for image classification, object detection, facial recognition, and medical imaging.
💬 NLP Used for sentiment analysis, speech recognition, translation, and question answering.
✨ Generative AI Used for chatbots, text generation, image generation, and content creation tools.
